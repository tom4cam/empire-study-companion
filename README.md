# Code of Ethics — Study Companion

A mobile-friendly study aid (reader, summaries, flashcards, practice quizzes)
for real estate continuing-education topics, starting with the NAR Code of Ethics.

This is a **study aid only**. It does not complete or grant CE credit — finish
the official course with your CE provider for credit.

Static site (vanilla HTML/CSS/JS). Open `index.html` via any static host.
